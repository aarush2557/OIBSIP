# 🚗 Car Price Prediction using Machine Learning

## 📌 Overview

This project predicts the selling price of used cars using Machine Learning regression algorithms. It includes data preprocessing, feature engineering, exploratory data analysis (EDA), model training, evaluation, and feature importance analysis.

---

## 🎯 Objective

Build a regression model that predicts the selling price of a used car based on features such as:

- Car Brand
- Car Age
- Kilometers Driven
- Fuel Type
- Seller Type
- Transmission
- Owner Type

## 🏆 Results

The models were compared using MAE, RMSE, and R² Score. The best-performing model was selected based on the highest R² score and lowest prediction error.
