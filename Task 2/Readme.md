# 📊 Unemployment Analysis with Python

## 📌 Project Overview

This project is part of the **Oasis Infobyte Data Science Internship (OIBSIP)**.

The objective is to perform **Exploratory Data Analysis (EDA)** on India's unemployment dataset to identify regional and temporal trends, with special emphasis on understanding the impact of the **COVID-19 pandemic** on unemployment rates.

---

## 🎯 Objective

* Analyze unemployment trends across different states and regions of India.
* Study monthly variations in unemployment rates.
* Compare unemployment rates before and after the COVID-19 outbreak.
* Visualize important insights using Python data visualization libraries.

---

## 🛠️ Tech Stack

* Python
* Pandas
* NumPy
* Matplotlib
* Seaborn
* Jupyter Notebook

---

## 📂 Dataset

**Dataset:** Unemployment in India

The dataset contains information such as:

* Region (State/UT)
* Date
* Estimated Unemployment Rate (%)
* Estimated Employed
* Estimated Labour Participation Rate (%)
* Area (Urban/Rural)

---

## 📈 Exploratory Data Analysis

The project includes:

* Data loading and inspection
* Missing value analysis
* Data cleaning and preprocessing
* Data type conversion
* Statistical summary
* Region-wise average unemployment analysis
* Monthly unemployment trend analysis
* Time-series visualization for major states
* Top 10 states with the highest average unemployment rate
* Correlation heatmap
* Pre-COVID vs. Post-COVID unemployment comparison

---

## 📊 Visualizations

The notebook contains multiple visualizations, including:

* 📌 Region-wise Average Unemployment Rate
* 📌 Monthly Unemployment Trend
* 📌 Time-Series Analysis of Major States
* 📌 Top 10 States by Average Unemployment Rate
* 📌 Correlation Heatmap
* 📌 Pre-COVID vs Post-COVID Comparison

---

## 📌 Key Insights

* Unemployment rates vary significantly across different states.
* Several regions consistently report higher unemployment than others.
* Monthly unemployment trends show noticeable fluctuations.
* The COVID-19 pandemic led to a substantial increase in unemployment rates.
* Labour participation and employment exhibit meaningful relationships with unemployment.
