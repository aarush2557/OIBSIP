# 🌸 Iris Flower Classification

## 📌 Objective
Build a machine learning model to classify Iris flowers into three species:
- Setosa
- Versicolor
- Virginica

- Jupyter Notebook
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Scikit-learn

- Dataset Shape
- Data Types

- Descriptive Statistics
- Pairplot
- Boxplots
- Correlation Heatmap

## 🤖 Machine Learning Models
- Logistic Regression
- K-Nearest Neighbors (KNN)

## 📈 Evaluation Metrics
- Accuracy Score
- Confusion Matrix
- Classification Report

## 🏆 Best Model
K-Nearest Neighbors (KNN) achieved the best performance with excellent accuracy on the Iris dataset.
